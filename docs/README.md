# ESP32-S3 Modbus RTU - Documentation

## Project Overview

This project provides ESP-IDF v5.4.4 support for ESP32-S3 with integrated Modbus RTU, WiFi, and MQTT functionality.

**Author**: Qwythos (Empero AI)  
**Framework**: ESP-IDF v5.4.4

## Quick Start

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd esp32s3-modbus-rtu
   ```

2. **Build the project**:
   ```bash
   idf.py build
   ```

3. **Flash and monitor**:
   ```bash
   idf.py flash -p /dev/ttyUSB0 monitor
   ```

## Components

### Modbus RTU
- **Protocol**: Modbus RTU over serial
- **Modes**: Master (client) only
- **Configuration**: 
  - Baud rate: 9600
  - Data bits: 8
  - Parity: None
  - Stop bits: 1

### WiFi
- **Modes**: STA (station) by default
- **Features**: 
  - Wi-Fi scanning
  - DHCP client
  - HTTP client integration

### MQTT
- **Broker**: tcp://broker.empero.org:1883
- **Features**:
  - Client connection management
  - Publish/Subscribe
  - QoS levels 0-2
